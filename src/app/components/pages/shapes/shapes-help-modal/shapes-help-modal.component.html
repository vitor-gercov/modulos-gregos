Os shapes são padrões de digitação que se repetem ao longo de todo o braço. Para usá-los, você precisa saber onde está a tônica no instrumento. Este guia mostra como encontrar qualquer nota usando as cordas abertas como ponto de partida.

Cordas abertas — afinação padrão

Na afinação padrão, cada corda solta produz uma nota de referência. A partir dela, cada traste adicional sobe 1 semitom.

Trastes e semitons

1 traste = 1 semitom

Cada traste corresponde a 1 semitom. No 12º traste você encontra a mesma nota uma oitava acima — e o ciclo se repete.
