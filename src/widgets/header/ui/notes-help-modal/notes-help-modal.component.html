Notas naturais

São as 7 notas básicas da escala musical, sem alterações. Formam a base de toda a harmonia e melodia.

A = Lá
B = Sí
C = Dó
D = Ré
E = Mi
F = Fá
G = Sol

Acidentes

São símbolos que alteram a altura de uma nota natural, elevando ou abaixando sua afinação em meio tom.
